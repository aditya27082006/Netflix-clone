# 🎬 Netflix Clone

A responsive **Netflix Clone** built to replicate the look and feel of Netflix's landing page. This project was created to practice modern web development concepts, improve frontend development skills, and gain hands-on experience with building responsive user interfaces.

## 🚀 About the Project

This project recreates the Netflix homepage with a clean and responsive design. It focuses on implementing a visually appealing UI while following good HTML, CSS, and JavaScript practices.

The main objective of this project was to strengthen my understanding of:
- Responsive web design
- Modern HTML5 and CSS3
- JavaScript fundamentals
- Flexbox and CSS Grid layouts
- Creating pixel-perfect user interfaces

## ✨ Features

- 🎥 Netflix-inspired landing page
- 📱 Fully responsive design for desktop, tablet, and mobile
- 🎨 Modern and clean UI
- 🖼️ Hero banner section
- 📺 Multiple content sections
- ❓ FAQ section
- 📧 Email signup section
- 🔻 Footer with useful links
- ⚡ Smooth layout and styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Netflix-clone.git
```

2. Navigate to the project folder

```bash
cd Netflix-clone
```

3. Open `index.html` in your browser.

Or use **VS Code Live Server** for a better development experience.
